# VLSI Internship – Task 2

## Introduction to Verilog HDL and RTL Design of Basic Combinational Circuits

### Objective

This project demonstrates the implementation and simulation of basic combinational circuits using Verilog HDL.

### Implemented Modules

- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- Half Adder
- Full Adder

### Software Used

- EDA Playground
- Icarus Verilog
- EPWave
